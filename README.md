# cube-tuimer

A TUI cube timer.

| Keys                 | Action                    |
|----------------------|---------------------------|
| <kbd>Space</kbd>     | Next                      |
| <kbd>Backspace</kbd> | Cancel solve              |
| <kbd>R</kbd>         | Rescramble                |
| <kbd>C</kbd>         | Toggle colored background |
| <kbd>Q</kbd>         | Quit                      |
